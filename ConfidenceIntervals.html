Confidence Interval Calculations

marshall brown

05/20/2014

Documentation for the procedures used to calculate risk measure \((1-\alpha)*100\%\) confidence intervals.

Standard intervals

We sample \(B\) bootstrap replications to obtain \(\hat{\theta}_b\) for \(b = 1, 2, ..., B\). Where \(\hat{\theta}_b\) is calculated by refitting the model using the \(x_{(b)}\) and using \(x_{(b)}\) to estimate the measure.

Normal theory

Confidence interval calculated using normal quantiles and the bootstrap estimate of standard error:

\[ \hat{\theta} \pm z^{(1-\alpha/2)}se(\hat{\theta}_{(b)}) \]

Quantile bootstrap

Confidence interval calculated using the quantiles of the empirical bootstrap distribution of \(\hat{\theta}_b\).

\[ [\hat{\theta}_{(b)} ^{(\alpha/2)}, \hat{\theta}_{(b)} ^{(1-\alpha/2)}] \]

Bias corrected intervals

Bias-corrected quantile bootstrap

To obtain a bias-corrected estimate of \(\theta\), we sample \(B\) bootstrap replications and calculate \(bias_{boot}(\hat\theta) = (B^{-1} \sum_b \hat{\theta}^*_{(b)} ) - \hat\theta\) where, for each bootstrap replicate, \(\hat{\theta^*_b}\) is calculated by fitting the model using the bootstrap data \(x_{(b)}\) to obtain model coefficients \(\hat{\beta}^*_{(b)}\) and estimating the measure using the original data \(x\) given \(\hat{\beta}^*_{(b)}\).

The bootstrap bias-corrected (bc) estimate is then:

\[ \begin{aligned} \hat{\theta}^{bc} & = \hat\theta - bias_{boot}(\hat\theta) \\ & = 2\hat\theta - (B^{-1} \sum_b \hat{\theta}^*_{(b)} ) \\ \end{aligned} \]

To obtain confidence intervals for \(\hat{\theta^{bc}}\), we use double bootstrapping. For each bootstrap replicate, we (double) bootstrap again to calculate \(\hat{\theta}^{bc}_{(b)}\). We then use the \((\alpha/2, 1-\alpha/2)\) quantiles of the bootstrap distribution of \(\hat{\theta}^{bc}_{(b)}\) for the confidence interval:

\[ [\hat{\theta}_{(b)} ^{bc (\alpha/2)}, \hat{\theta}_{(b)} ^{bc (1-\alpha/2)}] \]

Here is some psuedo code to help be clear:

marker data x, statistic T(x, coefs)

eta_hat <- model.fit(x)
theta_hat <- T(x, coefs = eta_hat)

#coefficients from the bootstrap, but evaluated with orginal data x
#to be used to bias correct theta_hat
theta_star <- numeric(B) 

#double bootstrap is used to obtain a bias corrected version of theta_boot
#use the quantiles of this vector to obtain CI
theta_boot_bc <- numeric(B)

for( b in 1:B) {
  #bootstrap
  x.boot <- x[sample.int(n, replace = TRUE), ]  
  
  #coefficients from the bootstrap sample
  eta_boot <- model.fit(x.boot)
  
  #coefficients from the bootstrap, but evaluated with original data x
  theta_star[b] <- T(x, coefs = eta_boot)
  
  ##double bootstrap ##
  
  #same as theta_star, but one bootstrap world down
  theta_star_boot <- numeric(dblB)
  for(bb in 1:dblB){
    
    #bootstrap the bootstrap
    x.dblboot <- x.boot[sample.int(n, replace =TRUE),]
    
    #coefficients from the double bootstrap sample
    eta_dblboot <- model.fit(x.dblboot)
    
    #coefficients from the double bootstrap, 
    #but evaluated with bootstrap data x
    theta_star_boot[bb] <- T(x.boot, coefs <- eta_dblboot)
    
  }  ##end double bootstrap##
  
  #coefficients from the bootstrap, and evaluated with the bootstrap data
  theta_boot <- T(x.boot, coefs = eta.boot)
  
  #double bootstrap estimate of bias
  theta_boot_bias  <- mean(theta_star_boot) - theta_boot
  
  #bootstrap distribution of bias-corrected estimates of theta
  theta_boot_bc[b] <- theta_boot - theta_boot_bias 
  
}  ##end bootstrap

#bootstrap estimate of bias
theta_bias <- mean(theta_star) - theta_hat

#bootstrap bias-corrected estimate of theta
theta_bc <- theta_hat - theta_bias

#confidence intervals are quantiles of theta_boot_bc
ci <- quantile(theta_boot_bc, probs = c(alpha/2, 1-alpha/2))

Divine bias-corrected quantile bootstrap

Since in simulations we know the true measure values, we can calculate the divine bias-corrected quantile bootstrap. Which is the same as the quantile bootstrap above, but we bias-correct the sampling distribution of \(\hat{\theta}\) to be centered on the true measure value \(\theta\).

First we calculate the bias, using the simulated sampling distribution of \(\hat\theta\) across \(S\) simulations.

\[ bias_{divine}(\hat\theta) = S^{-1} \sum_s \hat\theta_s - \theta \]

The confidence interval is then:

\[ [\hat{\theta}_{(b)} ^{(\alpha/2)} - bias_{divine}(\hat\theta) , \hat{\theta}_{(b)} ^{(1-\alpha/2)} -bias_{divine}(\hat\theta)] \]
