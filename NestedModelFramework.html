Model

We simulate data from a logistic model with linear terms: \(Risk(X) = Pr(D=1 | X) = expit(\beta X)\) so that our fitted model is correctly specified. Markers are generated from multivariate normal distribution with marginal \(\sigma=1\) and pairwise correlations \(0.1\).

Simulations are ran with sample sizes set at n = 250, and n = 500.
